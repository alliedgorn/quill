# Lesson: Propose Proactive Design Work During Dry Spells

**Date**: 2026-03-30
**Source**: Session 20 — third consecutive empty board
**Context**: No design tasks assigned for 3 sessions. All pending items blocked on Gorn/external input.

## The Lesson

When the board is empty for multiple sessions, don't just watch and rest. Propose proactive design work to Zaghnal — UI audits, component pattern documentation, preparatory specs for known upcoming features (Forge Phase 2, splash screen, quick-repeat meals). A designer sharpens tools between battles.

## Why It Matters

Passive waiting creates session overhead with no output. The rest-sooner norm handles the immediate case, but across multiple sessions the pattern suggests a gap in the pipeline, not just timing. Proposing work keeps the design function visible and ready.

## How to Apply

After 2+ empty board sessions, DM Zaghnal with specific design proposals rather than waiting for tasks to appear.
