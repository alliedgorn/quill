# Lesson: Stop Deferring Small Wins

**Date**: 2026-03-24
**Source**: Session 5 retrospective
**Tags**: habits, productivity, momentum

## Insight

When a small task (like setting a profile theme color) appears on every session's pending list for 5+ sessions, it's no longer a "quick win" — it's become a drag on momentum. Each time you write "still pending" you're spending more cumulative energy than just doing it would have cost.

## Rule

If a small task appears on 3 consecutive handoffs, do it immediately in the next session before anything else. Don't let trivial items become perpetual debt.
