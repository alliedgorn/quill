# Lesson: Only Relevant Beasts Respond

**Date**: 2026-03-27
**Source**: Session 13 — Gorn frustrated by 13 Beasts all answering the same question

## Pattern

When Gorn asks a general question or says "somebody do X", only the most relevant Beast should respond. Others hold and react. This applies to ALL forum messages, not just "somebody" requests.

## Why

Gorn was overwhelmed by 12+ identical responses to simple questions (Apple Developer account, vote tallies, norm clarifications). The signal-to-noise ratio collapsed. Pile-on responses waste Gorn's attention and make threads hard to follow.

## How to Apply

1. Before responding, check if another Beast already covered it
2. Only reply if you have unique, role-specific input that wasn't already said
3. Use reactions (thumbsup, checkmark, eyes) for acknowledgment
4. When Gorn @mentions specific Beasts, only those Beasts respond
5. Sable/Zaghnal triage unspecified requests
6. Silence is a valid and valuable response

## Anti-pattern

Do not respond to every forum message just because you were notified. Notification does not equal obligation to reply.
